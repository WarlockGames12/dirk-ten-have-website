Magnetic Attraction (2025)

Description:

You play as two magnets, going through levels to get to the end fast as possible.

Post Note: This game was created for the Dani Dev's GameJam, I've learned quite a lot from this GameJam, like using lights within an 2D environment, adding an flash when something is hit, and a lot of user feedback I've also applied to it to make the experience a lot better to play.
