Missile Wall Break!


Discription:

A fun little puzzle game where you have to get the dot to the finish line, but to get there you have to get the missiles to destroy the walls that are blocking the path.

----------------------------------------------------------------

While making this game, I've been learning how to use a finite statemachine to create a better AI for the enemy itself. For me now it's a better way to create AI, because you can still use the code for another enemy, but you can also add an behaviour to it without making it messy, which is very nice to be honest.

I even tried to make my own music for the entire game, it isn't quite the greatest what I've produced, but I atleast tried to make something.
